# Getting Started with Git

```bash
# Create a new directory named "Git" and change into it.
mkdir Git; cd Git

# Configure the global user email for Git.
git config --global user.email "canihugyouonetime@gmail.com"

# Configure the global username for Git.
git config --global user.name "Al"

# Initialize a new Git repository in the current directory.
git init

# Set the default branch name for new Git repositories to "main".
git config --global init.defaultBranch main

```
